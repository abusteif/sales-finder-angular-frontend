Create Alerts to Track Your Favorite Items

After signing up, you can create alerts to get notified instantly when items go on sale. Never miss a great deal again!

To start tracking items and receiving alerts, you'll need to create a free account with {{ appName }} first.

Once you're signed up, you can create alerts in two ways:

  • Go to the Alerts page to create an alert with free text search
  • Click the item options menu (three dots) on any item to create an alert for that specific product
